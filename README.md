# webpack-es6-configs
Webpack ES6 Config files.

## Build folder structure
```
build
│   index.html
│   stats.json    
└───assets
    ├───js
    │   │   app.js
    │   │   dependencies.js
    │   └───maps
    │        │ app.map
    │        │ dependencies.map
    └───css
        │   styles.css
```

## Dependencies
- Webpack
- Path
- Lodash' object assign
- Html-Webpack-Plugin
- Compression-Webpack-Plugin
- Stats-Webpack-Plugin
